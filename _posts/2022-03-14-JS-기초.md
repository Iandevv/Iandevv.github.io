### 1. 콘솔   
> console.log( )   
> ( ) 안에 숫자 또는 문자열 등을 넣어 출력   
- 예 ) console.log(6); , console.log('add');
 
### 2. let, const +  변수
 > 변수에 이름을 지정하여 let 또는 const로 정의하여 값을 저장   
- 예 ) let num = 6; , const color = 'Blue';

### 3. 숫자
> 정수, 소수 등 모든 숫자를 포함   
- 예 ) 0, -1, 2, 3.456
