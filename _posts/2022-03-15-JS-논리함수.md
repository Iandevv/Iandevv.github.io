### 1. AND (&&)   
> 전체 문장이 참이 되어야 출력한다. 한 문장이라도 거짓이라면 출력하지 않는다.   
```
const password = prompt("Enter your password")

if (password.length > 6 && password.indexOf(" ") === -1) {
    console.log("Success!!");
} else {
    console.log("Fail!!");
}
```

### 2. OR (||)   
> 전체 문장이 참이 되어야 출력한다. 한 문장이라도 거짓이라면 출력하지 않는다.   
```
const age = 6;

if (age < 8 || age >= 65) {
    console.log("FREE");
} else {
    console.log("$10");
}
```

### 3. NOT (!)   
> 전체 문장이 참이 되어야 출력한다. 한 문장이라도 거짓이라면 출력하지 않는다.   
```
const age = 10;

if (!(age >= 0 && age < 8) || age >= 65) {
    console.log("Not Free!!");
}
```
