### 1. 배열
> [ ] 안에 숫자 또는 문자열을 나란히 줄을 세운 것을 말한다.   

```
let number = [1, 2, 3, 4, 5, 6];
let color = ['red', 'bule', 'pink', 'green'];
```

### 2. 배열 인덱스
> 문자열과 마찬가지로 배열에도 인덱스가 적용   

```
const color = ['bule', 'black', 'pink', 'green'];

color[1] = 'black'
color[3] = 'green'
```
