### 1. 화살표 함수 개요

```
const greet = (name) => {
    return `Hey ${name}!`;
};
```

### 2. 화살표 함수의 반환   
> 인수가 하나일 경우에만 ( ) 생략 가능 

```
const add = function(num) {
    return num + num;
};

const add = (num) => {
    return num + num;
};

const add = num => {
    return num + num;
};

const add = num => (
    num + num
);

const add = num => num + num;
```
